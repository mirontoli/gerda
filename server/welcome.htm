Välkommen till Gerda Programmet.

Vi skapar detta program för att hjälpa personer som gympar ordentligt. Programmet är mycket användbar och lätt att använda.

Varför skulle man skriva övningar på mangå paper istället för att använda teknology? Idag har de flesta en mobil som funkar med Java och vi tänkte att vi skulle underlätta spara övningar på varje pass.

Mobilen kan användas på gymmet så enkelt. Man behöver bara mata in vilka övningar skall köras med kilo, sets och antal. Programmet kopplar till webbsidan där alla informatioon sparas automatisk och kartläggas.

Det finns också andra webbsidor där man kan få mer information eller kan köpa utrustning.
